# Agent Documentation

This directory contains reference documentation for AI coding agents working on the Elaro project.

## Contents

- **apex-review-checklist.md** - Comprehensive Apex code review checklist covering security, performance, and best practices
- **appexchange-security.md** - AppExchange security review requirements and guidelines
- **breaking-changes.md** - Salesforce platform breaking changes to watch for (v65.0+)
- **governor-limits.md** - Salesforce governor limits reference and optimization strategies

## Purpose

These documents provide agents with:
- Deep technical context about Salesforce development constraints
- Security and compliance requirements for AppExchange
- Platform-specific gotchas and best practices
- Quick reference for governor limits and optimization

## Usage

Agents should reference these documents when:
- Reviewing code for security vulnerabilities
- Preparing for AppExchange security review
- Optimizing code for governor limits
- Understanding breaking changes in new API versions

## Maintenance

Update these files when:
- New Salesforce API versions introduce breaking changes
- AppExchange security requirements change
- Governor limits are updated
- New best practices emerge from the Salesforce community
